# Project Technology Stack

This document outlines the core technologies used in the frontend of the Connectio project.

## Core Technologies
- **React + Vite**: Frontend framework and build tool.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **shadcn/ui**: Reusable component system.

## State Management & Data Fetching
- **Zustand**: For local and global UI state management.
- **React Query**: For server state management and caching.
- **Axios with interceptors**: For HTTP requests and API communication.

## Forms & Validation
- **React Hook Form + Zod**: For flexible form handling and schema-based validation.

## UI/UX Enhancements
- **Framer Motion**: For animations and page transitions.
- **Lucide React**: For scalable and consistent SVG icons.
- **Sonner**: For toast notifications.
- **Recharts**: For composable and responsive analytics dashboards.

## Real-time & Authentication
- **Native WebSocket / Socket.io**: For real-time chat and updates.
- **JWT authentication with HttpOnly cookies**: For secure user authentication.
